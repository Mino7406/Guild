# 길드(Guild Bot)

몬스터 헌터 유저들을 위한 디스코드 생태 정보 데이터베이스 봇입니다. 수렵에 떠나기 전, 몬스터의 약점, 육질, 소재 드랍률, 상태이상 정보 등을 디스코드 명령어 한 번으로 간편하게 검색할 수 있습니다.

## 주요 기능 (Features)
- **몬스터 정보 검색 (`/몬스터`)**: 몬스터의 기본 정보, 서식지, 위험도, 파괴/절단 가능 부위를 깔끔한 UI로 제공합니다.
- **육질 및 약점 확인**: 타격 부위별 육질(절단/타격/탄) 및 속성 약점을 직관적으로 보여줍니다.
- **소재 드랍 정보 (상위/하위)**: 몬스터 수렵 시 획득 가능한 소재를 상위와 하위 탭으로 나누어 제공합니다.
- **상태이상 및 유효 아이템**: 독, 수면, 마비 등의 상태이상 유효도(⭐⭐⭐)와 구멍 함정, 섬광탄 등의 아이템 통용 여부를 안내합니다.
- **스마트 버튼 & 도배 방지**: 해당 몬스터에 없는 데이터 탭은 자동으로 숨기며, 10초 쿨타임 시스템(나만 보기 경고창)으로 서버 도배를 완벽히 방지합니다.

## 기술 스택 (Tech Stack)
- **Language**: JavaScript
- **Data Format**: JSON (GSON 라이브러리를 통한 데이터 파싱)
